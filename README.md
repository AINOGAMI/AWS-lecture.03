# AWS-lecture.03/

・　APサーバーの名前とバージョンは

・　Puma version: 5.6.4 

・　サーバー終了時は「コントロール+C」

・　^C- Gracefully stoppingと出てくれば停止

・　再接続は「bundle exec rails s」で出来ました。


・　DBサーバーの名前とバージョンは

・　mysql  Ver 8.0.30 for Linux on x86_64 

・　MYSQLログイン前だと、「mysql --version」で調べられる

・BDサーバー起動と確認は　「sudo service mysqld start && sudo service mysqld status」

・サーバーの接続終了するときは「exit」
